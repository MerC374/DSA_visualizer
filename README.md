# DSA Visualizer ⚡

An interactive sorting algorithm visualizer with real-time animations.

## Algorithms
- Bubble Sort — O(n²)
- Selection Sort — O(n²)
- Insertion Sort — O(n²)
- Quick Sort — O(n log n)

## Features
- Real-time step-by-step animation
- Color coded bars (comparing, swapping, sorted)
- Speed control slider
- Live comparison and swap counter
- Sorting logic written in C++

## Tech Stack
C++, HTML, CSS, JavaScript

## How to run
Open index.html in any browser — no installation needed!
